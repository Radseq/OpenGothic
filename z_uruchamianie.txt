Jak użyć

Na czystej bazie PostgreSQL:

createdb gothic_mmo

Potem:

psql "postgresql://USER:PASSWORD@HOST:5432/gothic_mmo" \
  -v ON_ERROR_STOP=1 \
  -f db/migrations/postgres/production/001_gothic_mmo_production_schema.sql

Walidacja:

python tools/check_postgres_mmo_schema.py \
  --dsn "postgresql://USER:PASSWORD@HOST:5432/gothic_mmo"





Jak uruchomić

Najpierw migracje:

psql "$DATABASE_URL" -v ON_ERROR_STOP=1 \
  -f db/migrations/postgres/production/001_gothic_mmo_production_schema.sql

psql "$DATABASE_URL" -v ON_ERROR_STOP=1 \
  -f db/migrations/postgres/production/002_bootstrap_import_pipeline.sql

Potem najpierw wygeneruj SQL na sucho:

python tools/import_runtime_sqlite_to_postgres.py \
  --sqlite runtime/g2notr.sqlite \
  --dry-run-sql /tmp/import_g2notr.sql

Jak wygląda dobrze, wykonaj import:

python tools/import_runtime_sqlite_to_postgres.py \
  --sqlite runtime/g2notr.sqlite \
  --dsn "$DATABASE_URL" \
  --realm-key local-dev \
  --account-name local-import \
  --character-key PC_HERO

Walidacja:

python tools/check_postgres_bootstrap_import.py \
  --dsn "$DATABASE_URL" \
  --realm-key local-dev \
  --character-key PC_HERO






















Jak odpalić

Najpierw baza:

sudo apt update
sudo apt install mysql-server mysql-client

sudo systemctl enable --now mysql

Wejdź do MySQL:

sudo mysql

Utwórz DB i usera:

CREATE DATABASE gothic_mmo
  CHARACTER SET utf8mb4
  COLLATE utf8mb4_0900_ai_ci;

CREATE USER 'gothic'@'localhost' IDENTIFIED BY 'gothic_dev_password';

GRANT ALL PRIVILEGES ON gothic_mmo.* TO 'gothic'@'localhost';

FLUSH PRIVILEGES;

Wyjdź:

exit
  Potem:

mysql --default-character-set=utf8mb4 \
  -u gothic -p gothic_mmo \
  < db/migrations/mysql/production/001_gothic_mmo_production_schema.sql

  -u gothic -p gothic_mmo \
  < db/migrations/mysql/production/002_bootstrap_import_pipeline.sql
